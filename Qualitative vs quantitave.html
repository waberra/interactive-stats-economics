üìù Qualitative Data (Categorical)

Definition: Data that describes qualities, characteristics, or categories

Key Features: Cannot be measured numerically in a meaningful way

Types:

  • Nominal: Categories with no natural order (colors, names, brands)
  • Ordinal: Categories with natural order (ratings, grades, rankings)

üî¢ Quantitative Data (Numerical)

Definition: Data that can be measured or counted using numbers

Key Features: Mathematical operations make sense

Types:

  • Discrete: Countable whole numbers (students, cars, goals)
  • Continuous: Measurable values (height, weight, time)

üéØ Key Questions to Ask:

  • Can you do math with it? If yes ‚Üí Quantitative, If no ‚Üí Qualitative
  • Does it describe a quality/characteristic? If yes ‚Üí Qualitative
  • Can you measure or count it meaningfully? If yes ‚Üí Quantitative
  • Would averaging make sense? If yes ‚Üí Quantitative, If no ‚Üí Qualitative
