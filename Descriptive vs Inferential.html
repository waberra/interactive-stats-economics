üìä Descriptive Statistics

Purpose: Summarize, organize, and describe data that has been collected

Key Features: Present facts about the data, no generalizations beyond the data

Examples: Calculating averages, creating graphs, finding percentages, measuring spread

üîç Inferential Statistics

Purpose: Make predictions, generalizations, or conclusions about a population based on sample data

Key Features: Draw conclusions beyond the immediate data, test hypotheses, make predictions

Examples: Hypothesis testing, confidence intervals, regression analysis, predictions

üéØ Key Distinctions:

  • Descriptive: "What happened?" - Summarizing existing data
  • Inferential: "What can we conclude?" - Making predictions or generalizations
  • Descriptive: Uses words like "average," "percentage," "shows," "displays"
  • Inferential: Uses words like "predict," "conclude," "estimate," "test," "significant"
  • Descriptive: No uncertainty involved - describing what we observe
  • Inferential: Involves uncertainty and probability - making educated guesses
