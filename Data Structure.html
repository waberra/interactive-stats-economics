üìä Cross-Sectional

Definition: Data collected at one point in time across different entities

Structure: Multiple units, single time period

Example: Provincial GDP in 2023

üìà Time Series

Definition: Data for one entity collected over multiple time periods

Structure: Single unit, multiple time periods

Example: Canadian inflation 2010-2023

üìã Panel Data

Definition: Multiple entities observed over multiple time periods

Structure: Multiple units AND multiple time periods

Example: Provincial unemployment 2020-2023

üéØ Key Questions for Classification:

  • How many entities? One vs Many
  • How many time periods? One vs Many
  • Cross-Sectional: Many entities, One time ‚Üí Snapshot across units
  • Time Series: One entity, Many times ‚Üí Track changes over time
  • Panel Data: Many entities, Many times ‚Üí Both dimensions

Decision Matrix:

  • One entity + One time = Single observation (not a dataset)
  • Many entities + One time = Cross-sectional
  • One entity + Many times = Time series
  • Many entities + Many times = Panel data
